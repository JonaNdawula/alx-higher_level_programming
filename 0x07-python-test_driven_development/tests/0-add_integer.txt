0-add_integer.txt
====================


Import the 'add_integer' function from the 'example' module:

>>> from example import add_integer

use it to add the two integers:

>>> add_integer(2,3)
5

Can be also used with floats:

>>> add_integer(7.5, 4.3)
11

if either 'a' or 'b' is not an integer or a float, raise a TypeError:

>>> add_integer(10, "11")
Traceback (most recent call):
	...
TypeError:  b must be an integer

>>> add_integer("11.9", 2)
Traceback (most recent call):
	...
TypeError: a must be an integer
